# hot water tank

pure front-end Canvas2D mini-game with beat-reactive visuals (Web Audio API + `AnalyserNode`).

## run locally

from this folder:

```bash
python3 -m http.server 8000
```

open `http://localhost:8000` and click **Play**.

## audio file

put an mp3 at `assets/hot-water-tank.mp3` (recommended), or change `AUDIO_CANDIDATES` in `main.js`.

## controls

- `WASD` move
- `E` interact (valve, console, torches, switches)
